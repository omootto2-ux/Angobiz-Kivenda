Política de Privacidade

1. Dados Recolhidos

Recolhemos dados pessoais como nome, e-mail, telefone e localização para prestar os nossos serviços e melhorar a sua experiência.

2. Uso dos Dados

Os seus dados são utilizados para: processar encomendas, enviar comunicações relevantes, melhorar a plataforma e cumprir obrigações legais.

3. Partilha de Dados

Não vendemos os seus dados a terceiros. Partilhamos apenas com parceiros necessários para a prestação do serviço (ex.: transportadoras, processadores de pagamento).

7. Retenção de Dados
